# clashConfig
clash custom config

## 配置文件 / Configuration Files

### nodnsleak.ini
防止DNS泄露的Clash配置文件，无分流群组。

This configuration file prevents DNS leaks in Clash with no traffic split groups.

**主要特性 / Key Features:**
- 解决DNS泄露问题 / Prevents DNS leaks
- 使用ACL4SSR规则集 / Uses ACL4SSR rulesets
- 支持广告拦截 / Ad blocking support
- 自动节点选择 / Automatic node selection
- 直连中国大陆网站和IP / Direct connection for mainland China sites and IPs

**使用方法 / Usage:**
使用此配置文件链接作为Clash订阅转换的配置参数。

Use this configuration file URL as the config parameter for Clash subscription conversion.
